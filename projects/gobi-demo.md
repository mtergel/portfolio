# Gobi

#### E-commerce demo page

### Stuff used to make this project

- React (NextJS)
- Chakra UI
- downshift
- swiper

### Description

Fully responsive E-commerce demo page based on Gobi. WAI-ARIA compliant search component. Uses swiper for mobile sliders.
